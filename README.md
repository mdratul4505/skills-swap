# Skill Swap

## Project Theme
An interactive platform for individuals to offer, learn, and trade skills within their local area. Whether it’s guitar lessons, language exchange, coding help, or yoga training — users can browse listings, rate experiences, and connect with local skill providers.


## Project Description
SkillSwap is a platform where users can explore, learn, and practice a variety of skills ranging from music, coding, fitness, language learning, creative arts, and cooking. The platform provides a seamless user experience with authentication, private routes, and profile management.

## Key Features
- User authentication with Firebase
- Google Sign-In
- google Sign-Up
- Private Routes for protected pages
- password and email Sign-In
- Skill categories with animated cards
- Edit and update user profile
- Responsive UI using Tailwind CSS & DaisyUI


## Technologies & Packages

- React Router
- Firebase
- Tailwind CSS
